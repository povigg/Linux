# Linux Cheat Sheat

4 command types:

1.  **An executable command**
2. **A command built in into the shell**
3. **A shell function**
4. **An Alias**

* **type** command - display cmd type
* **which** cmd - display executable location
* **help** cmd

 **rm -r** - remove directory and its content
